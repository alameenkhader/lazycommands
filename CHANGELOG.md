## [0.1.0] - 2025-12-19

### Added
- Initial release
- Sequential command execution with visual progress tracking
- Terminal UI with status indicators
- Support for command-line arguments and stdin input
- Fail-fast behavior on command errors
- Debug logging to file
- Cross-platform support (macOS, Linux, Windows)
- Version information embedded in binary
